Reproducing Typewriter

Work in Progress

An early 20th-century typeface designed for readability at lower point sizes. It was a typeface you'd use with Linotype machines and whatnot to simulate typewritten letters. A few computer books also used it, including the Turbo Pascal 3.0 manual.

At a size of 12pt, Reproducing Typewriter is a pica font, 10 characters per inch, and 6 lines per inch. It should also be suitable for screenplay writing.

send(to, from, count)
register short *to, *from;
register count;
{
        register n=(count+7)/8;
        switch(count%8){
        case 0:	do{	*to = *from++;
        case 7:		*to = *from++;
        case 6:		*to = *from++;
        case 5:		*to = *from++;
        case 4:		*to = *from++;
        case 3:		*to = *from++;
        case 2:		*to = *from++;
        case 1:		*to = *from++;
                }while(--n>0);
        }
}
